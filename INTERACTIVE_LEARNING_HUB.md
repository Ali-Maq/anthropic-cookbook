## üöÄ Getting Started

1. **Set up your environment**:
   ```bash
   pip install anthropic jupyter pandas python-dotenv
   ```

2. **Get your API key**:
   - Sign up at https://console.anthropic.com
   - Generate an API key
   - Create `.env` file:
     ```
     ANTHROPIC_API_KEY=your_key_here
     ```

3. **Start with Module 1, Lesson 1.1**:
   ```bash
   jupyter notebook interactive_lessons/module_1/lesson_1.1.ipynb
   ```
